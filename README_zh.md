# ai-delivery

帮助非技术背景用户用 AI 做真实软件开发的项目合集。

不需要会写代码。你只需要清楚地知道自己想要什么。

## Skills

### dev-delivery

AI 驱动的完整开发交付流程——从理解你的需求开始，AI 先写出颗粒度最小的开发和测试规划，然后按规划一步步开发、逐个测试、集成点做回归测试，直到交付真实可用的结果。开发中途如果需要你来决策，AI 会暂停并给出清晰的选项清单，你决定后继续。

## 给非技术背景的用户

你不需要会写代码，只需要知道自己想要什么。

1. 安装 Claude Code（免费）
2. 从这个仓库下载 skill
3. 告诉 AI 你想做什么
4. AI 完成其余所有工作

### 你可以用它做什么

- 做一个简单的网站
- 做一个批量重命名文件的小工具
- 做一个个人记账工具
- 任何你能用语言描述清楚的想法

### 工作流程

1. **你告诉 AI** 想做什么
2. **AI 探索**项目，写出开发和测试规划
3. **AI 逐块开发**，每块都带测试
4. **AI 集成**所有模块，做回归测试
5. **你验证**最终结果

### 中途可以改需求吗？

可以。随时说"我想把 X 改成 Y"，AI 会调整。

### 项目卡住了怎么办？

AI 会给你一个清晰的决策清单，列出所有需要你决定的事情，每个选项写清楚优缺点。你选完告诉 AI，它继续。
