# ai-delivery

[**English version**](README.md)

*一组面向非技术人员的 AI 开发工具，让没有编程基础的人也能做出真实的软件项目。*

---

## 这是什么

这是一组帮你用 AI 做开发的 skill（工具）。

你不需要会写代码、不需要懂技术。只需要你知道自己想要什么——比如"我想做一个记账的小工具"或"我想做一个可以记录我每天工作内容的网站"。

然后把这件事告诉 AI，AI 会帮你把东西做出来。

---

## 能做什么

- 个人网站或博客
- 小型 Web 应用（如记账本、习惯追踪器）
- 自动化电脑上的重复操作
- 简单的聊天机器人或工作流自动化

---

## 需要准备什么

1. **一个清晰的想法** — 你想做什么，用大白话描述就行
2. **Claude Code** — Anthropic 出品的免费 AI 编程工具
3. **一台电脑** — Mac、Windows 或 Linux 都可以

---

## 工作流程

```
你描述你想要什么
        ↓
AI 生成开发规划（用你能看懂的话）
        ↓
你审核通过
        ↓
AI 按步骤开发和测试，每步都记录真实结果
        ↓
你验证结果 — 有问题 AI 马上改
        ↓
交付（或者一份等待你决策的清单）
```

| `dev-delivery` | 驱动 AI 完成完整开发周期：理解需求 → 规划 → 开发 → 测试 → 交付 |

## 怎么用

**第一步：** 从 [claude.ai/code](https://claude.ai/code) 下载 Claude Code（免费）

**第二步：** 在你的项目文件夹里运行：
```bash
git clone https://github.com/maxilen/ai-delivery.git ./.claude/
```

**第三步：** 告诉 Claude Code：
> "用 dev-delivery skill 帮我开发 [你的想法]"

然后 AI 会自动推进，你只需要审核和做决定。

---

## 常见问题

**Q：需要会写代码吗？**
不需要。用大白话描述你的需求就行。

**Q：开发过程中想改需求怎么办？**
直接说。AI 会调整并继续。

**Q：项目卡住了怎么办？**
AI 会给你列出一份"需要你决定的事情"，你告诉 AI 你选什么，AI 继续做。

**Q：做出来的东西不对怎么办？**
随时可以说"不对，我要改成 xxx"，AI 会重新做。
