# ai-delivery

> 一个没有编程基础的小白也能用 AI 做真实开发项目的 skill 合集

---

## 这是什么

这是一组帮你用 AI 做开发的工具（skills）。

不需要你会写代码，只需要你知道自己想要什么——比如"我想做一个网站"、"我想做一个手机 App"、"我想优化一下我现有的工具"。

AI 会帮你：

- 理解你想要什么
- 制定开发计划（不需要你操心技术细节）
- 真正把东西做出来
- 测给你看，告诉你哪里还需要你做决定

---

## 第一个 skill：dev-delivery

### 它能做什么

dev-delivery 是一个 AI 开发助手，能从"我想做 xxx"开始，一直到做出一个真实可用的东西。

它会：
1. 先搞清楚你想要什么
2. 自己做一份开发计划（拆到最小步骤）
3. 按计划开发，每一步都测给你看
4. 告诉你哪里卡住了、需要你做什么决定
5. 做完一个阶段会让你验收

### 适合谁用

- 想做一个网站/App/工具，但没有技术团队
- 不懂代码，但想清楚自己要什么
- 想要一个真实的交付物，不只是聊天

### 不适合谁

- 完全不知道自己要做什么（需要你自己想清楚需求）
- 想要的是现成软件不需要开发

## 怎么用

### 准备工作

你需要先安装 Claude Code（一个 AI 编程工具）：

1. **安装**：打开终端，运行：
   ```bash
   npm install -g @anthropic-ai/claude-code
   ```

2. **登录**：运行 `claude` 按提示登录（需要账号，可以用 GitHub 登录）

3. **拉取这个 skill**：
   ```bash
   # 在你项目目录下运行
   git clone https://github.com/maxilen/ai-delivery.git ./.claude/
   ```

### 开始你的项目

1. **想清楚你要做什么**（不需要技术术语，用大白话描述）
2. **打开终端，进入你的项目目录**
3. **运行 Claude Code**：
   ```bash
   cd 你的项目目录
   claude
   ```
4. **告诉它你想做什么**，比如：
   - "用 dev-delivery skill 帮我做一个个人简历网站"
   - "用 dev-delivery skill 帮我做一个客户管理表格"
   - "用 dev-delivery skill 帮我做一个自动回复邮件的工具"
5. **Claude 会问你一些问题**，如实回答就行
6. **AI 会开始工作**，你只需要：
   - 看它给你看进展
   - 回答它的问题
   - 验收它做出来的东西

## 你需要提供什么

AI 需要你提供的是**需求**，而不是技术方案。

### 好的需求（清晰、具体）：
- "我想做一个网站，让客户可以预约我的时间"
- "我需要一个表格，记录我所有客户的联系方式和跟进状态"
- "我想做一个自动工具，每天早上把昨天的销售数据发到我的邮箱"

### 不好的需求（模糊、无法执行）：
- "我想做数字化转型"
- "帮我做一个 AI 工具"
- "搞一个系统"

### 你需要配合的事情

开发过程中，AI 会停下来问你：

- "这个地方有两种方案，你选哪个？"（比如：简单免费但慢 vs 快速但收费）
- "你的预算是多少？"
- "你希望界面是什么风格？"

这些需要你来做决定，不用担心选错，AI 会解释每个选项的优缺点。

## dev-delivery 工作流程

```
你告诉AI想要什么
       ↓
AI 制定开发计划（你确认）
       ↓
AI 按计划开发（每个步骤都测给你看）
       ↓
遇到需要决策的地方 → AI 停下来问你
       ↓
你做决定 → AI 继续
       ↓
每个模块完成 → AI 展示测试结果
       ↓
所有模块拼在一起 → 做回归测试
       ↓
最终交付 → 你验收
```

## 常见问题

**Q：我完全不懂代码，能用吗？**
A：可以。这个 skill 就是设计给不懂代码的人用的。你只需要说你想要什么，AI 会处理技术细节。

**Q：做出来的东西能用吗？**
A：能。AI 会真实地开发和测试，不是只给你一个演示版本。

**Q：需要多少钱？**
A：Claude Code 使用 Anthropic 的 API，会消耗 token 费用。新用户有免费额度。具体费用取决于项目大小，一般小型项目可能几美元。

**Q：我不满意 AI 做的决定怎么办？**
A：随时可以说"不对，我要改成 xxx"。AI 会调整。

**Q：项目中途卡住了怎么办？**
A：告诉 AI "继续"，或者重新描述你的需求。AI 会基于之前的工作继续。
